# Real-Time High Speed Traffic Sign Detection

## Description
This project focuses on building a Convolutional Nueral Network (CNN) model to classify and detect different traffic signs for use in self-driving vehicles. 
We used the dataset from Kaggle, consisting of over 3,000 training images and 600 test images for a multiclass classification task. The dataset can be accessed at:
https://www.kaggle.com/datasets/pkdarabi/cardetection?select=car

## Dependencies
The following Python libraries and frameworks are required to run this notebook:
- `kaggle`
- `numpy`
- `pandas`
- `matplotlib`

## Setup
1. Add your Kaggle API credentials to the Google Colab Secrets (key icon on the left).
   Name them `KAGGLE_USERNAME` and `KAGGLE_KEY`.
2. Run the setup cells to autehnticate and download the dataset.
3. Run the data preparation cells to see the images with their labels.
